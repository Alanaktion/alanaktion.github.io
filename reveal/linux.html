Distributions

There are hundreds of Linux-based Operating Systems

Ubuntu

  • The most popular distribution
  • Very user-friendly
  • Based on Debian

Debian

  • Heavily-tested, stable distribution
  • Upstream for Ubuntu and hundreds of other distros
  • Fairly user-friendly

Arch Linux

  • Minimal Operating System
  • Open-ended setup, user builds the entire system from packages manually

OpenSuse

  • Feature-rich distribution
  • Primarily designed for business workstations

CentOS

  • Ryan's favorite
  • Based on the closed-source Red Hat

FOSS

Free and open-source software

Most Linux distributions are built primarily on Open Source softare, where the source code is public and can be contributed to by anyone.
